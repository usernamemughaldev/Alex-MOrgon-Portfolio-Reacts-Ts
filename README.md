## üé® About This Project

This portfolio is more than just a gallery; it's an interactive experience. It uses smooth animations and a clean, modern design to create a lasting impression.

### Key Features:

*   **Interactive 3D Background:** A captivating background powered by Three.js.
*   **Smooth Scrolling & Animations:** GSAP and Framer Motion provide a fluid user experience.
*   **Fully Responsive:** Looks great on all devices, from desktops to mobile phones.
*   **Modern UI Components:** Built with the excellent shadcn/ui and Radix UI libraries.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js (v18 or later recommended)
*   npm or yarn

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username/your_project_name.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Start the development server
    ```sh
    npm run dev
    ```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.
